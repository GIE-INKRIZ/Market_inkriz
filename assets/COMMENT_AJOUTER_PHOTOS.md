# Comment ajouter les photos de l'équipe

1. Renommez vos photos exactement comme suit :
   - tanguy.jpg
   - majid.jpg
   - louis.jpg
   - valentin.jpg

2. Glissez les fichiers dans ce dossier `assets/`

3. Faites un commit + push :
   git add assets/
   git commit -m "Ajout photos équipe"
   git push

Les photos apparaîtront automatiquement sur la page.
Format recommandé : JPG carré, 400x400px minimum.
